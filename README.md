# 🛍️ Ecommerce Web

Backend system provides RESTful API for web or mobile.

---

## 📖 Introduction

This Java/Spring Boot backend is designed to handle server-side logic, data processing, and communication for an **Ecommerce application** built with microservices architecture.

---

## ✅ Prerequisites

Before you begin, ensure you have the following installed:

- Java Development Kit **(JDK 17+)**
- **Maven** or **Gradle** build tool
- Database (e.g., `MySQL`, `PostgreSQL`, `MongoDB`)
- Liquibase for Database Migration
- **Spring WebFlux** for reactive programming
- **Kafka** (Zookeeper + Broker) for event-driven communication
- Docker (optional for containerization)
- Postman for API testing

---

## 🚀 Features

- ✅ Microservices Architecture
- ✅ Spring Boot 3+
- ✅ RESTful APIs
- ✅ Database Integration with **JPA/Hibernate**
- ✅ Reactive Programming with WebFlux
- ✅ Messaging with Kafka
- ✅ Docker Support
- ✅ Centralized Configuration and Service Discovery
